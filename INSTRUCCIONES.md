# Prueba Técnica: Dashboard de Monitoreo Ambiental

## Objetivo

Desarrollar una **aplicación web completa** (frontend + backend) para visualizar y analizar datos de puntos de toma de muestras ambientales.

---

## Requisitos Técnicos

### Stack obligatorio:

* **Frontend:** React con TypeScript
* **Backend:** Node.js/Express con TypeScript
* **Comunicación:** REST API

### Backend (API):

1. Crear endpoint `GET /api/samples` que sirva los datos del JSON proporcionado.
2. Implementar:
    * **Paginación** (10 muestras por página)
    * **Filtros**

### Frontend (Dashboard):

1. Mostrar las muestras en una **tabla con indicadores visuales**.
2. Implementar:
    * **Filtros avanzados**
    * **Sistema de paginación**
3. Los filtros deben **reflejarse en la URL** para compartir vistas.


## Entrega
### Requisito Adicional:

Se debe entregar un link a un repositorio de GitHub (público) con el código fuente completo del proyecto.
